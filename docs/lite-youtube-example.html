Note: This page demonstrates various ways to use the lite-youtube web component. All examples use real YouTube videos to show actual functionality.

1. Basic Video Embed

The simplest way to embed a YouTube video. Just provide the video ID.

<lite-youtube videoid="guJLfqTFfIw"></lite-youtube>

2. Video with Custom Title (Accessibility)

Adding a title improves accessibility and SEO.

<lite-youtube 
  videoid="dQw4w9WgXcQ" 
  videotitle="Rick Astley - Never Gonna Give You Up">
</lite-youtube>

3. Video with Fallback Link (Recommended)

Provides a fallback for users without JavaScript and improves SEO.

<lite-youtube videoid="jNQXAC9IVRw" videotitle="Me at the zoo">
  <a class="lite-youtube-fallback" 
     href="https://www.youtube.com/watch?v=jNQXAC9IVRw">
    Watch on YouTube: "Me at the zoo"
  </a>
</lite-youtube>
Watch on YouTube: "Me at the zoo"

4. Auto-load When Scrolled Into View

The video loads automatically when it becomes visible in the viewport.

<lite-youtube 
  videoid="9bZkp7q19f0" 
  videotitle="PSY - GANGNAM STYLE"
  autoload>
</lite-youtube>

5. Start Video at Specific Time

Start the video at 30 seconds.

<lite-youtube 
  videoid="kJQP7kiw5Fk" 
  videotitle="Luis Fonsi - Despacito"
  videoStartAt="30">
</lite-youtube>

6. Privacy-Enhanced Mode (No Cookies)

Uses youtube-nocookie.com for better privacy.

<lite-youtube 
  videoid="OPf0YbXqDm0" 
  videotitle="Mark Ronson - Uptown Funk"
  nocookie>
</lite-youtube>

7. High-Quality Poster Image

Use maximum resolution thumbnail (if available).

<lite-youtube 
  videoid="YQHsXMglC9A" 
  videotitle="Adele - Hello"
  posterquality="maxresdefault">
</lite-youtube>

8. Custom YouTube Parameters

Disable controls and related videos.

<lite-youtube 
  videoid="RgKAFK5djSk" 
  videotitle="Wiz Khalifa - See You Again"
  params="controls=0&rel=0">
</lite-youtube>

9. Custom Aspect Ratio (4:3)

Override the default 16:9 aspect ratio using CSS custom properties.

<lite-youtube 
  class="custom-ratio"
  videoid="2Vv-BfVoq4g" 
  videotitle="Ed Sheeran - Perfect">
</lite-youtube>

10. Flat Look (No Shadow Overlay)

Disable the gradient shadow overlay for a cleaner look.

<lite-youtube 
  class="no-shadow"
  videoid="CevxZvSJLk8" 
  videotitle="Katy Perry - Roar">
</lite-youtube>
